# OpenAI — Latest Model Lineup (as of March 17, 2026)

## Model Overview

### 🧠 Thinking/Reasoning Models

**GPT-5.4 Thinking** (Released: March 5, 2026) — ⭐ LATEST
- OpenAI's newest and most capable reasoning model
- Available in two variants: **GPT-5.4 Thinking** and **GPT-5.4 Pro**
- 33% reduction in factual errors vs GPT-5.2
- Focus on professional workflows
- **Not available to free-tier users** — paid only (Plus/Pro)
- Source: [Wikipedia — GPT-5.4](https://en.wikipedia.org/wiki/GPT-5.4)

**Previous reasoning models (for context):**
- **GPT-5.2 Thinking** (Dec 11, 2025) — Instant + Thinking + Pro modes; GPT-5.2-Codex for coding
- **GPT-5.1 Thinking** (Nov 12, 2025) — Added personalities, shopping research; Codex-Max for agentic coding
- **GPT-5 Thinking** (Aug 7, 2025) — First unified GPT-5 system with auto-routing between fast & deep models

**o-series reasoning models:**
- **o3** (Apr 16, 2025) — Standalone reasoning model, preceded by o3-mini (Jan 31, 2025); o3-pro (Jun 10, 2025)
- **o4-mini** (Apr 16, 2025) — Successor to o3-mini, multimodal (text + images), chain-of-thought reasoning
- ⚠️ The o-series appears to have been superseded by GPT-5.x Thinking models for ChatGPT users

### 💬 Standard (Fast/Instant) Models

**GPT-5.4** (Released: March 5, 2026) — ⭐ LATEST
- Likely includes an "Instant" mode like prior GPT-5.x releases (unconfirmed for 5.4 specifically)
- Not available to free-tier users

**GPT-5.2 Instant** (Dec 11, 2025) — Previous standard model
- Speed-optimized, non-reasoning mode
- Better at spreadsheets, financial modeling, presentations, multi-step execution

**GPT-5.1 Instant** (Nov 12, 2025) — Warmer personality, 8 personality options

### ⚡ Fast/Lightweight Models

**GPT-5 Thinking Mini / GPT-5 Thinking Nano** — Within the GPT-5 system
- gpt-5-main-mini (fast, high-throughput)
- gpt-5-thinking-mini (reasoning, smaller)
- gpt-5-thinking-nano (API only, even smaller/faster)

**GPT-4.1 mini / GPT-4.1 nano** (Apr 14, 2025) — API-focused lightweight models
- GPT-4.1 mini replaced GPT-4o mini for ChatGPT free users
- GPT-4.1 nano — smallest variant for developers

**o4-mini** (Apr 16, 2025) — Lightweight reasoning model with multimodal support

> ⚠️ It's unclear which exact "mini" model free-tier ChatGPT users currently get in Mar 2026 — likely GPT-5.x system with auto-routing, or GPT-4.1 mini fallback.

### 🎨 Image Generation

**GPT Image** (Released: March 2025)
- Replaced DALL-E 3 in ChatGPT
- Native image generation built into GPT models (not a separate model)
- Major improvement: much better text rendering in images
- Weaker for non-Latin alphabets
- Can generate images based on existing images (edit/modify)
- Source: [Wikipedia — ChatGPT](https://en.wikipedia.org/wiki/ChatGPT)

**DALL-E 3** — Legacy, no longer the default in ChatGPT (replaced March 2025)

### 🔓 Open-Source / Open-Weight

**GPT-OSS** (Aug 5, 2025)
- Set of two open-weight models with reasoning capabilities
- Released days before GPT-5 launch

### 🛠️ Specialized Models

- **GPT-5.2-Codex** — Coding-specialized variant
- **GPT-5.1-Codex-Max** — Agentic coding model (Nov 19, 2025)
- **Codex Agent** (May 2025) — Based on fine-tuned o3, runs in cloud VM or locally via API; writes software, runs tests, proposes PRs

## Pricing (as of March 2026)

| Tier | Price | Key Features |
|------|-------|-------------|
| **Free** | $0 | Limited GPT-5.x access (auto-routed), GPT-4.1 mini, basic image gen, rate-limited |
| **Plus** | $20/mo | Higher usage limits, GPT-5.4 access, GPT-5.x Thinking, Advanced Voice, Deep Research |
| **Pro** | $200/mo | Unlimited GPT-5.x, GPT-5.4 Pro, GPT-5 Thinking Pro (parallel compute), Deep Research, o3-pro |
| **Team** | Per-seat (enterprise) | Collaborative workspace, admin controls |
| **Enterprise** | Custom | Full enterprise features |
| **Go** (India) | ₹399/mo (~$5) | Higher limits than free, launched Aug 2025 |

### Free Tier Includes:
- Access to ChatGPT with GPT-5.x (limited messages/hour)
- GPT-4.1 mini
- Image generation (GPT Image)
- ChatGPT Voice
- Web browsing / search
- ⚠️ GPT-5.4 is **NOT** available to free-tier users

### Pro Tier ($200/mo) Includes:
- Unlimited GPT-5 / 5.x access
- GPT-5.4 Thinking + GPT-5.4 Pro
- GPT-5 Thinking Pro (parallel test-time compute — highest quality)
- Deep Research (o3-based comprehensive reports)
- ChatGPT Agent (multi-step tasks)
- o3-pro access
- Priority access to new features

## Key Talking Points

1. **GPT-5.4 is brand new** (12 days old as of today) — the "5.4 era" the user mentioned
2. **The o-series (o3, o4-mini) is effectively absorbed** into GPT-5.x Thinking models for consumer use
3. **GPT-5's architecture is unique**: auto-router that switches between fast and deep reasoning models transparently
4. **Image generation is now native** to GPT (GPT Image), not a separate DALL-E model
5. **Pricing hasn't changed**: Plus still $20/mo, Pro still $200/mo since Dec 2024
6. **Free tier is decent** but misses the latest GPT-5.4
